# 第１１回日本情報オリンピック D - パスタ (Pasta)

## 反省

```text
dp[i][j][k]: i日目まででi-2日目にパスタj, i-1日目にパスタkを作る方法のうち、3日連続して同じパスタを作らないような予定の総数
```

という方針はたてれたが、次の遷移の方針がたてれなかった。

```text
dp[i][j][k] → dp[i+1][k][l]
```

パスタlは3通り考えられるのでループを回す必要があるが、次の条件によって処理をスキップする必要があることに注意。

```text
・i日目のパスタが指定されている
・3日連続して同じパスタになる
```

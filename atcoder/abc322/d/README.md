## 具体的な方針

各ブロックで平行移動と90°回転を行い、グリッドに収まるブロックを保存しておく。
各ブロックの集合について全列挙して重ならないように#でグリッドを埋められる組み合わせがあればOK。なければNG
平行移動: 4×4のグリッド上での移動方法を全列挙((x,y) => (x-3~x+3, y-3~y+3))して#のマスのみ移動させる
90°回転: (x,y) => (y,3-x)をすべてのマスに適用させる

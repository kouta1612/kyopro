# atcoder

## atcoder cli

### コマンド一覧

abc001ディレクトリとその直下にコンテスト情報が作成される

```bash
acc new abc001
```

サンプルケースが正しいか確認する

```bash
oj t -c "go run ./main.go" -d tests/
```

実装したコードを提出する

```bash
acc submit main.go
```
